# API Specifications

**auto-dev-test-target-1** is a library (npm package), not a service. It does not expose HTTP/REST/GraphQL APIs. Its interface is a TypeScript/JavaScript module API consumed via `import` statements.

See [c4-container.md](../c4-container.md) for the full package API surface documentation.
